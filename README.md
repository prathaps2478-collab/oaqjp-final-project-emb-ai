# Emotion Detection Application using IBM Watson NLP

This repository contains the final project for the Emotion Detection Application developed using Python, Flask, and the IBM Watson NLP library.

## Project Overview

The application analyzes text entered by the user and detects emotions such as:

* Joy
* Sadness
* Anger
* Fear
* Disgust

The project demonstrates:

* Emotion detection using Watson NLP
* Python package creation
* Unit testing
* Error handling
* Flask web deployment
* Static code analysis using pylint

## Technologies Used

* Python 3
* Flask
* IBM Watson NLP
* HTML/CSS
* Pylint
* UnitTest
